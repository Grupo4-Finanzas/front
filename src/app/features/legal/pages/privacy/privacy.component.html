Política de Privacidad

En CrediAuto valoramos la seguridad y confidencialidad de la información del usuario. Esta política explica qué datos se registran, cómo se utilizan y cómo se protegen dentro del sistema.

1. Datos registrados

El sistema puede registrar datos como nombre completo, DNI, correo electrónico, contraseña hasheada, información del vehículo y parámetros financieros utilizados en las simulaciones.

2. Uso de la información

La información registrada se utiliza para autenticar al usuario, asociar simulaciones a su cuenta, generar cronogramas de pago, calcular indicadores financieros y permitir la consulta del historial.

3. Protección de credenciales

Las contraseñas no se almacenan en texto plano. El sistema guarda una versión hasheada para proteger la seguridad de la cuenta del usuario.

4. Historial y simulaciones

Cada simulación se almacena asociada al usuario autenticado. Esto permite revisar resultados anteriores, consultar cronogramas y comparar diferentes escenarios de financiamiento.

5. No uso comercial de datos

La información ingresada en esta versión del sistema se utiliza únicamente para fines académicos y funcionales del simulador. No se vende ni se comparte información personal con terceros.

6. Seguridad

El acceso a las funciones principales del sistema requiere autenticación mediante usuario y contraseña. Además, el sistema utiliza mecanismos de seguridad para proteger las rutas privadas.

7. Derechos del usuario

El usuario puede revisar su información de perfil, modificar sus datos disponibles y eliminar simulaciones registradas en su historial cuando la aplicación lo permita.
